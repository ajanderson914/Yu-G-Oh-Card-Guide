Project Purpose

Yu-Gi-Oh Card Guide is a simple web app for people who are new to Yu-Gi-Oh. The goal is to make card information easier to search, read, and understand.

Users can search for cards, view basic card details, and save cards they want to remember later.

Data Source

This project uses the YGOPRODeck API to get Yu-Gi-Oh card information. Saved cards are stored using a Supabase database.
