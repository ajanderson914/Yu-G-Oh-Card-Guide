# Yu-Gi-Oh Card Guide

## Description

Yu-Gi-Oh Card Guide is a simple web application designed to help beginners search for Yu-Gi-Oh cards and understand basic card information. Users can search for specific cards, view card names, images, types, and descriptions, and save cards to a personal saved cards list.

This project uses the YGOPRODeck API for card information and Supabase to store saved cards.

## Target Browsers

Use this in regular browsers, cant guarantee it will work in others:

- Google Chrome
- Safari
- Microsoft Edge
- Firefox
